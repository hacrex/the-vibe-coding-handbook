## 📋 Prerequisites

### Essential (Must Have)
- ✅ Basic computer literacy (file management, installing software)
- ✅ Internet connection for accessing AI tools

- ❌ Computer science degree
- ❌ Years of coding experience
- ❌ Expensive hardware (cloud options available)

## ✅ Getting Started Checklist

Before diving into the content, complete these setup steps:

### Step 1: Environment Setup
- [ ] Install [VS Code](https://code.visualstudio.com/)
- [ ] Install Git ([Windows](https://git-scm.com/download/win), [Mac](https://git-scm.com/download/mac), [Linux](https://git-scm.com/download/linux))
- [ ] Create a GitHub account (if you don't have one)
- [ ] Fork this repository to your account

### Step 2: AI Tools Setup
- [ ] Sign up for [Claude](https://claude.ai/) (recommended for coding)
- [ ] Sign up for [ChatGPT](https://chat.openai.com/) (backup option)
- [ ] Install GitHub Copilot (optional, free for students)
- [ ] Bookmark [Cursor IDE](https://cursor.sh/) (AI-native editor)

### Step 3: Learning Setup
- [ ] Create a dedicated folder for practice projects
- [ ] Set up a note-taking system (Notion, Obsidian, or simple markdown)
- [ ] Join our community Discord/Slack (link in Resources)
- [ ] Schedule 1-2 hours daily for learning

### Step 4: Mindset Preparation
- [ ] Accept that AI will make mistakes - verification is key
- [ ] Commit to understanding code, not just copying
- [ ] Embrace iterative development
- [ ] Prepare to share your journey (blog, Twitter, etc.)
