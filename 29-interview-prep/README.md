# Interview Preparation with AI

## Coding Interviews

### LeetCode Practice
```
Prompt: "Act as FAANG interviewer. Give me medium array problem.
After I solve, provide: correctness feedback, complexity analysis,
optimizations, similar problems."
```

### Pattern Recognition
Practice these patterns:
- Two Pointers
- Sliding Window
- Binary Search
- DFS/BFS
- Dynamic Programming
- Backtracking

## Behavioral Questions

### STAR Method
- **S**ituation
- **T**ask
- **A**ction
- **R**esult

### Top 10 Questions
1. Challenging technical problem
2. Conflict with teammate
3. Failure and lessons
4. Prioritizing deadlines
5. Proud project
6. Disagreement with manager
7. Changing requirements
8. Mentoring experience
9. Difficult stakeholders
10. Why this company?
